# NodeJS
API NodeJS criação de um crud de produtos.

# Comandos

1 - npm init -y
2 - npm install express
3 - npm install --save-dev nodemon
4 - npm install mongoose
5 - npm install require-dir
6 - npm install mongoose-paginate
7 - npm install cors


# Start do projeto

1 - npm run dev
2 - localhost:3001
3 - :)
